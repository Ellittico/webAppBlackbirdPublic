# BlackBird Sentinel WEB App
Web application that enables online access to core features for network monitoring, scanning, and analysis. It provides a frontend interface to visualize metrics, logs, and distributed data, along with detailed reports for each scan.

Future developments will include continuous monitoring and distributed anomaly detection powered by machine learning.
